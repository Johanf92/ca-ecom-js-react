# E-commerce Store

This is an e-commerce store built with React, Vite, and Tailwind CSS. The project showcases a modern, responsive online store where users can browse products, view details, add items to the cart, and proceed to checkout.

## Features

- **Product List**: View a list of products with images, descriptions, and prices.
- **Product Page**: View individual product details and add items to the cart.
- **Cart**: View items in the cart, adjust quantities, and proceed to checkout.
- **Checkout**: Complete the purchase process with order confirmation.
- **Search**: Real-time product search functionality.
- **Contact Form**: A working contact form with validation.

## Technologies Used

- **React**: Frontend JavaScript library for building the UI.
- **Vite**: A fast and modern build tool for React applications.
- **Tailwind CSS**: A utility-first CSS framework for custom styling.
